kenobiDB

Welcome

kenobiDB is a lightweight and high performance document-based data store that supports simple operations like insertion, update, removal, and search. Check out the source code on GitHub or view the project on the Python Package Index.

kenobiDB is Fun


from kenobi import KenobiDB

>>> db = KenobiDB('example.db')

>>> db.insert({'name': 'Obi-Wan', 'color': 'blue'})
True

>>> db.search('color', 'blue')

And Easy to Install


$ pip install kenobi
