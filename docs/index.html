kenobiDB

A stupid and small document based database, supporting very simple usage including insertion, removal and basic search, kenobiDB useses the pickle module built into the python standard library. Check out the source code on github/patx/kenobi or view the project on pypi

Use it

You can install kenobiDB using the pip command pip install kenobi      //     view documentation using pydoc pydoc kenobi

See it in action

>>> from kenobi import KenobiDB

>>> db = KenobiDB('database.json', auto_save=False)

>>> db.insert({'name': 'user1', 'groups': ['user']})
>>> db.insert({'name': 'user2', 'groups': ['admin', 'user']})
>>> db.insert({'name': 'user3', 'groups': ['sudo', 'user']})

>>> db.search('name', 'user1')
[{'name': 'user1', 'groups': ['user']}]

>>> db.find_any('groups', ['admin', 'sudo'])
[{'name': 'user2', 'groups': ['admin', 'user']},
 {'name': 'user3', 'groups': ['sudo', 'user']}]

>>> db.find_all('groups', ['admin', 'user'])
[{'name': 'user2', 'groups': ['admin', 'user']}]

>>> db.save_db()
True
