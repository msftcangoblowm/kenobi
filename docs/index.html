kenobiDB

A stupid and small document based database, supporting very simple usage including insertion, update, removal and basic search, kenobiDB useses the pickle module built into the python standard library. Check out the source code on github/patx/kenobi or view the project on pypi

Use it

You can install kenobiDB using the pip command
pip install kenobi

View documentation using pydoc
pydoc kenobi

See it in action

>>> from kenobi import KenobiDB

>>> db = KenobiDB('example.pkl', auto_save=True)

>>> db.insert({'name': 'user1', 'groups': ['admin', 'user']})
True

>>> db.search('name', 'user1')
[{'name': 'user1', 'groups': ['admin', 'user']}]
